<center>
  krissy是一款基于tauri+react+typescript开发的全自动deltarune翻译包安装程序  
</center>

## 如何使用  
1. 从release中下载最新的软件包，并解压到电脑上的任意位置  
2. 运行krissy.exe  
3. 将游戏根目录复制到选择游戏路径的对话框中
4. 选择您需要安装的翻译包
5. 选择是否创建备份
6. 选择可用的镜像站点（非必须选择）
7. 点击安装并等待
8. 完成安装

## 关于该项目  
该项目为deltarune好人汉化组的翻译包安装程序
